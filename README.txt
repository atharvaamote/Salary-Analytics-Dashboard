# Salary Analytics Dashboard

## Project Overview

An interactive Power BI dashboard built using a dataset of 250,000 job records to analyze salary trends across job roles, experience levels, industries, education levels, and work arrangements.

## Tools Used

- Python
- Pandas
- Power BI
- DAX

## Dashboard Features

- KPI Cards
  - Total Records
  - Average Salary
  - Total Industries
  - Total Locations

- Interactive Filters
  - Location
  - Industry
  - Remote Work

- Visualizations
  - Average Salary by Job Title
  - Average Salary by Experience
  - Average Salary by Education Level
  - Average Salary by Industry

## Key Insights

- AI Engineers earn the highest average salaries.
- Salary increases steadily with experience.
- Higher education levels correlate with higher compensation.
- Industry significantly impacts salary outcomes.
